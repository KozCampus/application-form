# közcampus — frontend

A SvelteKit application using:

- [SvelteKit](https://kit.svelte.dev/) with [Svelte 5](https://svelte.dev/) (runes)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [shadcn-svelte](https://www.shadcn-svelte.com/) component library
- [Axios](https://axios-http.com/) HTTP client
- [mode-watcher](https://mode-watcher.svecosystem.com/) for dark mode

## Development setup

### Installation

1. Make sure [Node.js](https://nodejs.org/) is installed.

2. Install the dependencies.

```bash
npm install
```

### Running the application

Start the development server.

```bash
npm run dev
```

### Building

To create a production build:

```bash
npm run build
```

Preview the production build with `npm run preview`.
